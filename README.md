# React Film Project

This is a movie management app built with React. You can add, delete, and filter movies by rating, name, and 3D format.

## Features:
- Add movies
- Delete movies
- Filter by name, rating, and 3D format
- Sort movies by name or rating
- Display top 3 highest rated movies

## Technologies Used:
- React
- JavaScript
- CSS
- Git
